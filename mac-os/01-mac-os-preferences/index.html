가장 기본적인 환경설정!

소프트웨어 업데이트

뭐든지 업데이트부터 하고 시작한다.

시스템 환경설정

Dock

  • 화면에서의 위치를 왼쪽으로 바꾸고 크기를 적당히 줄인다.
  • Dock에서 최근 사용한 응용 프로그램 보기를 끈다.
  • 확대를 켜고 적당히 키운다. 이쁘다.

Mission Control

  • Spaces를 최근 사용 내역에 따라 자동으로 재정렬를 끈다.

    • 안 하면 미션 컨트롤 켤 때마다 위치가 바뀐다.

  • 응용 프로그램으로 전환할 때, 응용 프로그램에 대해 윈도우가 열려 있는 Space로 전환를 끈다.

    • 끄더라도 필요할 경우 독의 아이콘을 더블클릭하면 스페이스가 전환된다.
  • 응용 프로그램별로 윈도우 그룹 생성을 켠다. 아이콘이 떠서 이쁘다.

Siri

  • 끈다.

Spotlight

  • 난 어차피 Alfred 쓰니까 키보드 단축키...에 들어가 Spotlight 검색 보기를 껐다.

언어 및 지역

  • 선호하는 언어는 한국어, 영어.
  • 시간 포맷의 24시간제를 켠다.

  • 하단의 고급...을 눌러 날짜, 시간에서 한 자릿수로 되어 있는 것들을 모조리 두 자릿수로 고친다.

    • 시간 나올 때 2020년 1월 1일 대신 2020년 01월 01일이 나와야 마음이 편하다.

손쉬운 사용

  • 디스플레이 → 커서에서 커서 크기를 적당히 키우고 마우스 포인터를 흔들어 찾기를 끈다.
  • 포인트 제어기 → 트랙패드 옵션...에서 드래그 활성화를 켜고 세 손가락으로 드래그하기로 바꾼다.

보안 및 개인 정보 보호

  • 잠자기 또는 화면 보호기 시작 *즉시* 암호 요구 하도록 바꾼다.
  • 화면이 잠겨 있는 동안 메시지 표시를 켜고 잠금 메시지 설정...을 눌러 이름과 전화번호를 써넣는다.
  • FileVault를 켠다.
  • 방화벽도 켠다.

블루투스

  • 메뉴 막대에서 Bluetooth 보기를 켠다. 에어팟 연결할 때 필요함.

키보드

  • 키 반복과 반복 지연 시간을 적당히 빠르고 짧게 조절한다.
  • Control Strip 사용자화...를 눌러 쓰지도 않는 시리 아이콘을 밖으로 빼낸다.
  • 보조 키...를 눌러 Caps Lock(⇪) 키를 ⎋ Escape로 바꾼다.
  • 텍스트 탭에서 모든 자동완성 기능을 끈다. 큰따옴표와 작은따옴표도 각각 "abc", 'abc'로 바꾼다.
  • 단축키 탭에서 키보드 탐색을 사용하여 컨트롤 간에 초점 이동을 켠다.

  • 단축키 탭에서 입력 소스를 클릭하고 이전 입력 소스 선택을 시프트(⇧) + 스페이스로 변경한다.

    • fn을 누른 상태로 시프트(⇧)와 스페이스를 누르면 된다.
    • 컨트롤(⌃) + 스페이스는 터미널 상에서 쓸 일이 많고, 커맨드(⌘) + 스페이스는 Alfred로 쓰고 있어서 어쩔 수 없다 🙃

트랙패드

  • 탭하여 클릭하기를 끄고 클릭 세기를 가볍게로 조절한다.

날짜 및 시간

  • 시계 탭에서 요일 보기와 날짜 보기를 켠다.

    • 여기선 자꾸 월, 일이 한 자릿수로 나오던데 바꾸는 법을 모르겠다.

공유

  • 컴퓨터 이름을 적당히 바꾼다.

메뉴 막대

  • 커맨드(⌘) 키를 누르고 아이콘을 클릭하면 이리저리 움직일 수 있다. 쓸모 없는 것들은 밖으로 빼면 제거된다.
  • 배터리 아이콘을 클릭하고 퍼센트 보기를 켠다.

원화 대신 백틱 입력하기

macOS 한글 키보드는 백틱(`) 자리에 원화(₩)가 매핑되어 있다.

~/Library/KeyBindings/DefaultkeyBinding.dict 파일을 만들고 다음을 입력한다:

{
  "₩" = ("insertText:", "`");
}

껐다 켜야 함.

연속으로 키 입력하기

터미널을 열고 다음을 입력한다:

defaults write -g ApplePressAndHoldEnabled -bool false

껐다 켜야 함.
